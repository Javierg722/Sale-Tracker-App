# Sale Tracker PWA v3

This version redesigns the iPhone experience around:
- compact lot cards
- tap-for-details
- a simpler wash review screen
- separate tools/import area

## Better mobile UX
Instead of trying to show the entire spreadsheet on iPhone, this version keeps the workbook logic but presents it in a cleaner mobile layout.

## Current support
- Add Buy
- Record Sale
- Partial sale split-lot workflow
- Wash Review
- JSON backup import/export

## Next logical upgrade
- XLSX spreadsheet import/export
- PDF upload with review
- editable detailed lot fields
