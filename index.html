The app now starts with your actual workbook data. Use spreadsheet import anytime you update the Excel file and want the app refreshed. Export creates a workbook copy with your current app data written into the trade-entry cells while keeping formulas in place.

Manual entry rules

  • New buy = new lot
  • Full sale = close the lot
  • Partial sale = keep one lot and add each sale to that lot's Sales History table
  • Spreadsheet import reads the trade table from the Wash Sale Tracker sheet and replaces the current app data

How this app works

Main screen: one-line lot rows grouped by sleeve.

Tap a lot: open a compact lot summary. Use Details for the full extended view.

Partial sale: the app keeps one purchase lot and adds each sale to a Sales History table inside that lot.

Wash Review: only actual wash-sale rows appear there.
